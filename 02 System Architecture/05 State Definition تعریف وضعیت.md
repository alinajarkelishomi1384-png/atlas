# State Definition
تعریف وضعیت (State)

---

## Objective (هدف)

تعریف مفهوم «State» به عنوان نمایش کامل وضعیت فعلی عامل (Agent) و محیط در یک لحظه مشخص از زمان.

State پایه و مرجع اصلی تمام فرآیندهای تصمیم‌گیری، هدایت، کنترل، یادگیری و پیش‌بینی در سیستم است.

---

## Definition (تعریف)

State مجموعه‌ای از تمام اطلاعات ضروری است که وضعیت فعلی عامل و محیط را در یک لحظه مشخص توصیف می‌کند.

این اطلاعات باید به اندازه‌ای کامل باشند که سیستم بتواند وضعیت کنونی را درک کرده، آینده را پیش‌بینی کند و بهترین تصمیم ممکن را اتخاذ نماید.

---

## State Structure (ساختار State)

State از دو بخش اصلی تشکیل می‌شود:

### 1. Internal State (وضعیت داخلی)

نمایانگر وضعیت داخلی عامل (Agent) است.

نمونه‌ها:

- موقعیت (Position)
- سرعت (Velocity)
- شتاب (Acceleration)
- جهت و وضعیت فضایی (Attitude)
- نرخ دوران (Angular Rate)
- انرژی یا باتری
- سلامت سامانه (System Health)
- وضعیت عملگرها

---

### 2. External State (وضعیت محیط)

نمایانگر وضعیت محیط اطراف عامل است.

نمونه‌ها:

- شرایط جوی
- سرعت و جهت باد
- موانع
- اهداف
- تهدیدها
- قوانین و محدودیت‌های محیط
- سایر عامل‌های موجود در محیط

---

## World State (وضعیت جهان)

World State ترکیب Internal State و External State است.

این مدل، کامل‌ترین نمایش از وضعیت سیستم و محیط را فراهم می‌کند و مبنای تصمیم‌گیری سیستم خواهد بود.

---

## Purpose (هدف استفاده)

State به سیستم اجازه می‌دهد:

- وضعیت فعلی را درک کند.
- محیط را تحلیل کند.
- تغییرات را تشخیص دهد.
- آینده را پیش‌بینی کند.
- بهترین تصمیم را انتخاب کند.
- عملکرد خود را ارزیابی کند.
- از تجربه‌های گذشته یاد بگیرد.

---

## Characteristics (ویژگی‌ها)

یک State مناسب باید:

- دقیق باشد.
- به‌روز باشد.
- قابل اعتماد باشد.
- قابل اندازه‌گیری باشد.
- قابل استفاده در تصمیم‌گیری باشد.
- قابلیت به‌روزرسانی در زمان واقعی (Real-Time) داشته باشد.

---

## Relationship to Other Components (ارتباط با سایر بخش‌ها)

- Sensor Layer داده‌های اولیه را تولید می‌کند.
- State Estimation وضعیت فعلی را تخمین می‌زند.
- World Model وضعیت جهان را نگهداری می‌کند.
- Decision Core بر اساس State تصمیم می‌گیرد.
- Guidance System مسیر مناسب را تولید می‌کند.
- Control Strategy فرمان‌های کنترلی را ایجاد می‌کند.
- Adaptive Learning با استفاده از تغییرات State سیستم را بهبود می‌دهد.

---

## Core Principle (اصل بنیادی)

هر تصمیمی که سیستم می‌گیرد، تنها به اندازه کیفیت State آن هوشمند خواهد بود.

بنابراین، کیفیت تصمیم‌گیری مستقیماً به کیفیت تخمین، نگهداری و به‌روزرسانی State وابسته است.
